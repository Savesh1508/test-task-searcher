

========== Описание ==========
Это одностраничное приложение, которое позволяет искать
пользователей по email и number из JSON файла.

========== Установка и запуск ==========

# Frontend
1. Перейдите в папку `frontend`
2. Установите зависимости:
######################
#    npm install     #
######################

3. Запустите приложение:
######################
#   npm run dev      #
######################

# Backend
1. Перейдите в папку `backend`.
2. Установите зависимости:
######################
#    npm install     #
######################

3. Запустите сервер:
######################
#    npm run server  #
######################

## Использование
1. Откройте приложение в браузере.
2. Введите email (обязательно) и number (опционально).
3. Нажмите кнопку "Submit" для отправки запроса на сервер.
4. Результаты поиска отобразятся под формой.

Примечания
- Валидация email и number реализована на фронтенде.
- Поле number имеет маску `12-34-56`
